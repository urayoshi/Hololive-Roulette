# 🎰 ホロライブルーレット

このアプリは、TSVファイルに登録されたホロライブタレントの一覧から、
ランダムにタレントを抽選できる簡単な **Flask製Webアプリ** です。

## 🌟 主な機能
- 回す回数を指定してランダム抽選
- 重複をON/OFFで切り替え可能
- 抽選結果をタレントカラー付きで表示
- 将来的に各タレントのYouTubeチャンネルへリンク可能（拡張予定）

## 🧩 使用技術
- Python 3.x  
- Flask  
- HTML / CSS / JavaScript  
- TSVデータの読み込み (csvモジュール使用)
